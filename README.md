# Костыли 🚀

Утилита для быстрой и автоматизированной подготовки трансляций в vMix. Позволяет в пару кликов менять названия команд, их цвета на табло, а также обновлять RTMP-сервер и ключ трансляции.

## 🎯 Что нового
* Автоматический парсинг файла `stream_keys.txt`.
* Автоматическая очистка названий команд от мусора (FC, LFC, LFK и т.д.).
* Умный подбор цвета команды на основе базы `teams_colors.json`, которую придется своими силами заполнить.
* Возможность отправлять команды на любой vMix в локальной сети (по IP и порту).

## 📥 Установка и запуск (для пользователей)
Вам **не нужно** устанавливать Python. Приложение собрано в один независимый `.exe` файл.

1. Скачайте **`kostyli.exe`**.
2. Распакуйте в любую удобную папку.
3. Убедитесь, что рядом с файлом `.exe` лежат два файла конфигурации:
   - `stream_keys.txt` — список матчей и ключей.
   - `teams_colors.json` — база цветов команд.
4. Запустите.

## 🛠 Как пользоваться
1. **Адрес vMix:** В приложении укажите IP и порт компьютера, на котором запущен vMix (например, `192.168.1.5:8088`).
2. Нажмите **Проверить**, чтобы убедиться, что vMix доступен.
3. **Выбор матча:** Выберите нужный матч из списка.
4. Нажмите **Отправить в vMix** (или просто дважды кликните по матчу).
5. Приложение само обновит табло (названия и цвета) и настройки стрима!
